Виды селекторов

Селектор по названию элемента - будут выбраны все элементы с таким названием (например теги)

Селекторы по ID - будут выбраны все ID с таким названием

Селекторы по классу - будут выбраны все элементы содержащие данный класс

Селектор нескольких классов - добавивши к элементу несколько классов будут использованы стили заданные каждому классу у этого элемента

Селектор всех элементов - при выброе "*" в стилях данный стиль будет добавлен всем элементам на странице
